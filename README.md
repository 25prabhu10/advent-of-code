# Advent of Code Solutions

Solution to the [Advent of Code](https://adventofcode.com/) problems in both JavaScript and Python.

## Running Solutions

To run a solution, `cd` into its directory and invoke the program:

```bash
# day 1
cd aoc01

# JavaScript
node index.js

# Python
python main.py
```

Environments:

- Node.js `v19`
- Python `v3.10`
